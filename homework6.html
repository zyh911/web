1.What are JavaScript browser sniffers?

JavaScript Browser Sniffer is a browser identifier written in JavaScript and released under the LGPL license. It will tell which browser, version and operating system you (the visitor) are using (it's like phpsniff, but in JavaScript).

2.Why are they so commonly used in javascripts?

  1. Javascript gives straightforward access to objects of the DOM.
  2. Javascript cannot read/write to the local disk, so it's safe.
  3. Browsers have a JavaScript interpreter built-in, no plug-in is required. ——It's convenient.
  4. JavaScript can bring pages alive without the overhead of large downloads (e.g. applets, Flash, etc).
